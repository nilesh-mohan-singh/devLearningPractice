! - Inserts Doctype html tag
> - nested
+ - sibling
* - sibling
{} - Inserting
# - Id
. - class

ul>li - inserts an li tag inside an ul tag
ul>li>a - Inserts a tag inside a li tag inside an ul tag
ul>li*3 - inserts 3 li tags inside an ul tag
div - inserts a div tag
div>p*3 - inserts 3 p tags inside a div tag
